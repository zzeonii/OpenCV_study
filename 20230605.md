# 파이썬 가상환경

- 파이썬 버전확인
  - $ python3 -V
- pip 버전확인
  - $ pip -V
- 가상환경 설치 (파이썬 버전에 맞춰서 ex 3.8)
  - $ sudo apt install python3.8-venv
- 가상환경 생성 python3 -m venv [가상환경이름]
  - $ python3 -m venv venv
- 가상환경 활성
- linux or macOS
  - $ source ./venv/bin/activate
- 가상환경 확인
  - (venv)$ python -V 
  - (venv)$ pip -V
- 가상환경 비활성
  - (venv)$ deactivate

# 가상환경 내에 openCV 설치
- 가상환경 활성화
  - $ source ./venv/bin/activate
- pip를 통해 openCV 설치
  - (venv)$ pip install opencv-contrib-python
- pip 설치 내용 확인(둘 중 하나)
  - (venv)$ pip list
  - (venv)$ pip freeze
- pip 의존성패키지를 내보내기
  - (venv)$ pip freeze > requirements.txt
- pip 의존성패키지 복원
  - (venv)$ pip install -r requirements.txt
- 파이썬 버전 지정하여 설치
  - pip install opencv-contrib-python==4.7.0.68
